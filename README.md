# NYC_Citibike_Challenge
## Overview:
This project was going through data from a 2019 database that focuses on Citibike data in New York and going through user information to see a correlation between different variables of the data.

## Results:
There were many interesting points to focus on in this data. One focal point was based on the time bikes were used which was under an hour, usually.
